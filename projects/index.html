Archivio

Questo link porta ad un canale Telegram contenente una serie di link che conducono ad altrettanti canali telegram.
Questi canali fungono da archivio dei miei progetti sviluppati nel corso del tempo, raggruppati per tipologia.

I vari canali/argomenti trattati sono
  • Programmazione: contiene file in formato .txt o con altre estensioni, contenenti i vari progetti dentro cartelle zippate
  • Reti : contiene i file con estensione .pkt (Cisco Packet Tracer) dei vari progetti realizzati, salvati con la data ed un eventuale descrizione
  • Realta' aumentata : contiene file di progetti sviluppati con Unity e altre app
  • Data science : contiene file di progetti sull'analisi dei dati e tutto ciò che riguarda machine learning e intelligenza artificiale
